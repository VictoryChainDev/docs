# Add Victory Chain to Metamask
## Using the Victory Hub Page
1. Navigate to our [Hub Page](https://testnet.hub.victorychain.io/)
2. Click the 
**Connect** button in the *Your Wallet* section, and you will be prompted to switch to the Victory Chain Testnet Network

{/* ![hubpage](../images/hub.png) */}

## Manually via Metamask
You can add the network manually via the following details:
- Network Name: Victory Chain Testnet
- RPC URL: https://testnet.rpc.victorychain.io/
- Chain ID: 199207251
- Currency Symbol: VICTRY
